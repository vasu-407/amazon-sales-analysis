# Amazon Sales Analysis 📊

## 📌 Project Overview
This project analyzes Amazon sales data from 2019–2024 to identify trends, top products, and revenue patterns.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Dataset
- Format: Excel (.xlsx)
- Contains sales transactions data

## 📈 Key Insights
- Sales trends over time
- Top-performing products
- Revenue distribution
